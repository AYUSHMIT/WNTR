Source code
===============

Testing
-----------

WNTR includes automated tests using the nose testing package.  Test are
run nightly using the Jenkins continuous build and test server 
at Sandia National Laboratories. 
https://jenkins.sandia.gov/view/TEVA/job/resilience_trunk_python2.7

Bug reports
-----------

Bug reports can be submitted to the WNTR trac site at 
https://software.sandia.gov/trac/resilience
